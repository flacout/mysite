    The two apllets are:

  • DNA and Protein Sequence alignment
    This application find similarities and gaps between two sequences of character. It is used usually in Bioinformatics to compare DNA or protein sequences. I design the algorithm myself, based on publications about local and global alignment (Needlman-Wunsch and Smith-Waterman algorithm).

  • Photo editor and manager
    This application is a web adaptation of a project that I realized during one of my classes. It is written in Javascript and use a library from Duke university for images management.

  • Single Digit Classifier
    This is a machine learning application for which I have trained separately a Support Vector Machine Classifier with the MNIST dataset of handwritten digits. Before processing, an image is transformed by Principal Component Analysis. The classifier then is able to predict a new digit written by a human.
